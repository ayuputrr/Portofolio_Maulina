Deskripsi Proyek

JKu (Jadwal Kuliah) adalah aplikasi dashboard pribadi berbasis web yang dirancang khusus untuk mahasiswa dalam mengatur jadwal kuliah mereka secara efisien.

Dibangun menggunakan HTML5, Tailwind CSS, dan JavaScript (ES6+), aplikasi ini menyimpan data secara lokal menggunakan localStorage tanpa backend eksternal.

Fitur Utama

  • Tambah, edit, dan hapus jadwal kuliah dengan mudah
  • Filter jadwal berdasarkan hari (Senin - Minggu)
  • Penyimpanan data otomatis di localStorage
  • Interface responsive dengan Tailwind CSS
  • Simulasi loading saat load awal
  • Split halaman (Beranda & Jadwal)

Teknologi & Tools

  • HTML5, CSS3, JavaScript (ES6+)
  • Tailwind CSS
  • Vite untuk development server
  • Vercel untuk deployment
