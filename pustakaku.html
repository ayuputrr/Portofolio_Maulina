Deskripsi Proyek

Pustakaku adalah aplikasi pengelolaan koleksi buku yang dirancang untuk mencatat buku yang dimiliki, sedang dibaca, atau ingin dibeli. Aplikasi ini memiliki antarmuka interaktif, pencarian cepat, dan menggunakan Firebase untuk sinkronisasi data real-time.

Fitur Utama

  • CRUD buku dengan form validasi
  • Filter status: dimiliki, dibaca, ingin dibeli
  • Pencarian buku berdasarkan judul/penulis
  • Statistik jumlah buku per kategori
  • Realtime database (Firebase)
  • Local storage untuk filter sementara
  • Testing dengan React Testing Library
  • Penggunaan React Hooks dan PropTypes
