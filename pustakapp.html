Deskripsi Proyek

Pustaka-App adalah aplikasi web peminjaman buku fullstack yang menggunakan React di frontend dan Pyramid di backend. Aplikasi ini memudahkan pengguna untuk meminjam buku secara online, dengan pengelolaan data buku dan peminjaman oleh admin dan user.

Dependensi Paket

  • Frontend: react, react-router-dom, axios, testing-library, web-vitals
  • Backend: pyramid, pyramid_jinja2, sqlalchemy, alembic, waitress, pytest

Cara Menjalankan

  • Backend: pip install & run dengan pserve development.ini --reload
  • Frontend: npm install lalu npm start (di folder src)

Fitur Utama

  • Login user & admin
  • Dashboard terpisah
  • CRUD buku
  • Peminjaman & manajemen peminjaman
  • Favorit buku
  • Laporan admin

Testing Backend

  • Jalankan dari direktori backend: pytest
